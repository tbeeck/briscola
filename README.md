# Briscola

Implementation of the Italian card game [Briscola](https://en.m.wikipedia.org/wiki/Briscola).

Comes with a mix task to try it out in the command line!

## Installation

```elixir
def deps do
  [
    {:briscola, "~> 0.1.0"}
  ]
end
```

## Play

```bash
mix briscola.play
```
