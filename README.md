# Briscola

Implementation of the Italian card game [Briscola](https://en.m.wikipedia.org/wiki/Briscola).

Comes with a mix task to play on the command line!

## Installation

```elixir
def deps do
  [
    {:briscola, "~> 0.1.0"}
  ]
end
```

## Play in the command line

```bash
mix briscola.play
```
